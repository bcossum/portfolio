Projects

Board Game Tracker

A web application designed to help users track board game plays. Users can create accounts, curate their own favorite games, and add plays for any number of board games. I made use of boardgamegeek.com's API to pull from their database rather than relying on the user to create entries for the games they wished to create plays for. Eventually I will be adding in the ability to add other users as friends allowing you to check on each other's recent plays.
Challenges: Working with Board Game Geek's API was a considerable challenge. Their api is actually in XML, so I spent a lot of time trying to figure out how to even read it myself before I was able to get my code to parse the input and turn it into something a site user could understand.

Metroidvania Video Game

This was a group project I did with some class mates at Coding Dojo. We used the pygame library to create a short video game.
Challenges: The trick I found with game design was knowing where in the never ending loop that is the game to write your code. We tried very hard to keep our code organized and our method naming conventions make sense as to when the code written within would run whether that be when the game started, ended, every loop, or even only every couple of loops.
